.vim
====
My vim setup and plugin(s).

Install
=======
cd ~
git clone https://github.com/xnceth/.vim.git ~/.vim
ln -s ~/.vim/.vimrc ~/.vimrc
cd ~/.vim
git submodule init
git submodule update

Suitable vim version
====================
VIM - Vi IMproved 7.4 (2013 Aug 10, compiled Feb 19 2014 10:23:40)
