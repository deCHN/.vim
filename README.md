.vim
====
My vim setup and plugin(s).

Install
=======
cd ~<br/>
git clone https://github.com/xnceth/.vim.git ~/.vim<br/>
ln -s ~/.vim/.vimrc ~/.vimrc<br/>
cd ~/.vim<br/>
<br/>
git submodule update --init --recursive<br/>
